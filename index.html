A 3-week study plan

Week 1 — Foundations & design. Domains 1–2. Build one small end-to-end Claude app. Nail model selection trade-offs, prompt/context engineering, and prompt caching.

Week 2 — Integration & quality. Domains 3–4. Stand up a RAG pipeline; add tool use / MCP; write evals; practice diagnosing failures (retrieval vs. prompt vs. model).

Week 3 — Govern, communicate, operate. Domains 5–7. Guardrails, least privilege, HITL, compliance; stakeholder communication and lifecycle; Claude Code / developer enablement. Then take the full mock twice.
